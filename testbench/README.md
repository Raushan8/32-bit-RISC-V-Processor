# Testbench Files

This directory contains the Verilog testbenches used to verify the functionality of the RTL modules in the processor.

Each testbench applies different input stimuli, monitors outputs, and validates the behavior of the corresponding hardware module through simulation.

The testbenches are intended to ensure the correctness of individual modules as well as the complete processor before hardware implementation.

Simulation tools such as AMD Xilinx Vivado can be used to execute these testbenches and observe waveform outputs for verification and debugging.

# RTL Source Files

This directory contains the Register Transfer Level (RTL) implementation of the 32-bit Single-Cycle RISC-V Processor written in Verilog HDL.

Each module is designed to perform a specific function within the processor datapath or control path, following a modular and reusable design methodology.

Typical modules include:

* Program Counter (PC)
* Instruction Memory
* Register File
* ALU (Arithmetic Logic Unit)
* ALU Control
* Control Unit
* Immediate Generator
* Data Memory
* Multiplexers
* Branch Logic
* Top-Level CPU Module

These modules are integrated to implement the RV32I single-cycle processor architecture.
